$resourceFactory ( services in module ngSeed )

Description

A service used to generate and register resources.

Methods

  • builder($resource, $httpOptions)

    Generates an HTTP Resource.

    Parameters

    • $resource – {$resource} –

      ngResource

    • $httpOptions – {$httpOptions} –

      The HTTP Options object.

    Returns

    {resource} –

    The actual resource for resourceName.

  • ResourceFactory(resourceName)

    Generates a factory function that uses ngResource to talk to a REST resource that can be directly used with factory.

    Example of use with factory: 

    services
  .factory('ResourceService', ResourceFactory('resource'));

    Parameters

    • resourceName – {String} –

      The name of the resource.
