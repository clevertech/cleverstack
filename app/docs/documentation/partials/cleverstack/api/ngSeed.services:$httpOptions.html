$httpOptions ( services in module ngSeed )

Description

This options can be plugged in the $httpProvider or the $http object to set up application wide configuration for HTTP requests.

Examples

myApp.controller('Test', ['$scope', '$httpOptions', function ($scope, $httpOptions) {
  if( /localhost/.test($httpOptions.domain) ) {
    // you are working with your local server
  }
}]);
