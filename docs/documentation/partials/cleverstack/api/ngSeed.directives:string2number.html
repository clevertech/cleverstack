string2number ( directives in module ngSeed )

Description

Converts a string to a number. Useful in type="number" input elements that bind to a stringified number model.

Usage

as attribute
<ANY string2number>
   ...
</ANY>
as class
<ANY class="string2number">
   ...
</ANY>

Methods

  • fromField(number)

    Parameters

    • number – {Number, String} –

      Just the number that has been input.

    Returns

    {Number} –

    The number.

  • toField(number)

    Parameters

    • number – {Number, String} –

      Just the number that has been input.

    Returns

    {Number} –

    The number or 0.
